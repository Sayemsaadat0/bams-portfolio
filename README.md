# Install

Starter pack for next js with typescript project along with tanstack query , axios and shadcn 

## Clone 

Use the package manager  to install This.

```bash
Git clone <this is repo url>
```


## Installation

Use the package manager  to install This.

```bash
npm install
```

## Usage
To run this

```
npm run dev / yarn run dev / pnpm run dev
```
